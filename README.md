# breedy GitHub Pages upload

## 업로드 파일
GitHub 저장소 루트에 `index.html`을 업로드하면 됩니다.

## 접속 주소 예시
- 사용자 앱: `https://yivesanto.github.io/breedy/`
- 관리자 화면: `https://yivesanto.github.io/breedy/#admin`

화면 우측 하단 플로팅 버튼으로 사용자 앱과 관리자 화면을 전환할 수 있습니다.

## 현재 구현 범위
- 정적 UI 데모
- 사용자/관리자 화면 전환
- 관리자 화면 내부의 상태 변경 데모
- 새로고침 시 입력/변경 데이터 초기화

## 아직 연결되지 않은 기능
- 회원 로그인 및 관리자 권한 인증
- 사용자 데이터와 관리자 데이터의 실제 공용 DB 연동
- 사진 파일 업로드/삭제/저장
- 게시물, 개체, 리스팅, 장소의 실제 등록

실서비스 전환 시 Supabase 또는 Firebase 같은 인증·DB·스토리지 연동이 필요합니다.
